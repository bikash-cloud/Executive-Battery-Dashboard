# 🔋 Executive Battery Dashboard

An enterprise-grade Battery Management System (BMS) developed using **ESP32**, **Firebase Realtime Database**, and a **live Executive Web Dashboard**.

The system monitors four battery cells in real time, performs battery health analysis, detects abnormal operating conditions, uploads telemetry to the cloud, and visualizes live battery analytics through a professional dashboard.

---

# 🚀 Features

- Real-time 4-cell voltage monitoring
- Battery Health Score calculation
- Pack Voltage calculation
- Average Cell Voltage
- Cell Imbalance Detection
- Runtime State Detection
- Risk Analysis
- Relay Control
- Buzzer Alarm
- 16x2 LCD Status Display
- Firebase Realtime Database Integration
- Live Executive Dashboard
- Historical Voltage Trend
- Diagnostics Panel
- Fault History
- Operator Recommendation System

---

# 🛠 Hardware

- ESP32 DevKit V1
- 16x2 I2C LCD
- Relay Module
- Active Buzzer
- 4 × Potentiometers (Battery Cell Simulation)
